Components

Alerts

Sometimes a message needs to stand out from the crowd, Alerts provide that capability.

Sample Alert Styling

This is a default alert
This is a blue alert with margin top
This is an orange alert with margin top, left and right with border radius
Sample Code
<div class="row gutter-12">
<div class="alert">
<This is a default alert
</div>
<div class="alert blue margin-top-20">
<This is a blue alert with margin top
</div>
<div class="alert orange border-radius-10 margin-top-20 margin-left-50 margin-right-50">
<This is an orange alert with margin top, left and right with border radius
</div>
</div>

Ability to change the colour to any in the default big dipper theme by adding classes (orange, yellow, blue, red, green, pink, white). Ability to control margin all around the element as well as border radius
