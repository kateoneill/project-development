Containers

To contain content to various breakpoints or to have it fluid utilise constellation UIs built in container classes

Sample Containers

container-fluid : max-width 100%


xl-container : max-width 1200px


lg-container : max-width 960px


md-container : max-width 720px;


sm-container : max-width 480px


xs-container : max-width 100%


Sample Code
<div class="container-fluid">
<h3 class="Roboto grey2 align-center"> xl-container : max-width 100% </h3>
</div>
<div class="xl-container">
<h3 class="Roboto grey2 align-center"> xl-container : max-width 1200px</h3>
</div>
<div class="lg-container">
<h3 class="Roboto grey2 align-center"> lg-container : max-width 960px</h3>
</div>
<div class="md-container">
<h3 class="Roboto grey2 align-center"> md-container : max-width 720px</h3>
</div>
<div class="sm-container">
<h3 class="Roboto grey2 align-center"> sm-container : max-width 480px</h3>
</div>
<div class="xs-container">
<h3 class="Roboto grey2 align-center"> xs-container : max-width 100%</h3>
</div>
