At our core

Dark Mode

With Dark Mode Design at the heart of all elements of Constellation UI are designed with the dark theme in mind

Responsive

Constellation UI is built mobile first with a full 8-col grid system to allow developers resize columns for all screens

Customisable

Constellation UI doesn’t use pre-styled components, instead it offers developers options to customise every element

Key Features of Constellation UI

Layout

Employs a full grid system with containers and Constellations trademark “Platforms” for elevated content, essential for dark mode

Element

Customs classes for a variety of HTML elements: Buttons, Tables, Lists and Images

Component

A website is only a sum of its parts. Things like Dropdowns, Cards and Alerts as offered by Constellation UI make a website unique and offer a valuable user experience, Constellation provides custom classes for each of these elements for efficient development and sleek end-product.
