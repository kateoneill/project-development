Layout

Grid

At the foundation of any framework is a way to make it repsonsive. Make the content readable and presentable for all devices. Constellation Ui has created an 8-col grid to help you in these situations

Sample Grid

.xs-4-wide .md-2-wide

.xs-4-wide .md-2-wide

.xs-4-wide .md-2-wide

.xs-4-wide .md-2-wide

Sample Code
<div class="container-xl">
<div class="row">
<div class="xs-4-wide md-2-wide"> </div>
<div class="xs-4-wide md-2-wide"> </div>
<div class="xs-4-wide md-2-wide"> </div>
<div class="xs-4-wide md-2-wide"> </div>
</div>
</div>

Constellation UI is fully responsive. There are a variety of classes that allow as much customisability as possible. Constellation UI is an 8-col grid system. A row has 8 columns in a row. How the user chooses to organise those is completely up to them.

If you want a sidebar do 2-wide and 6-wide columns. Want a variety of cards, do 4 2-wide columns. There's an abundance of options and variety for users to include.

All the different breakpoints have their own classes. It is a mobile first framework so it works upward. If you want your framework to be full width on mobile devices but 50-50 on laptop size simply add the class xs-12-wide md-4-wide

For maximum customisability there are classes available to add a variety of gutter widths: 0, 12px and 24px. The default gutter width is 0px. To add gutter add a declaration to the row as follows:

Sample Grid

.xs-12-wide .md-6-wide
.xs-12-wide .md-2-wide
Sample Code
<div class="xl-container">
<div class="row gutter-12">
<div class="xs-8-wide md-6-wide"> </div>
<div class="xs-8-wide md-2-wide"> </div>
</div>
</div>

Order classes

xs-4-wide md-2-wide md-order-3 xs-order-2

xs-4-wide md-2-wide md-order-1 xs-order-3

xs-4-wide md-2-wide order-1 md-order-4 xs-order-4

xs-4-wide md-2-wide order-3 md-order-3 xs-order-1

Sample code
<div class="container-xl">
<div class="row">
<div class="xs-4-wide md-2-wide md-order-3 xs-order-2"> </div>
<div class="xs-4-wide md-2-wide md-order-1 xs-order-3"> </div>
<div class="xs-4-wide md-2-wide md-order-4 xs-order-4"> </div>
<div class="xs-4-wide md-2-wide md-order-3 xs-order-1""> </div>
</div>
</div>

Order classes allow you reorganise all tiles and rows on all sizes by adding the class for whatever size breakpoint you want followed by where in the order you want it to fall.
