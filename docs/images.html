Images

A picture says a thousand words and Constellation UI want to enhance the image experience by providing easy to use classes for all your image needs

Sample Image Styling

Sample Code
<div class="row gutter-12">
<div class="xs-4-wide">
<img class="fit-width border-radius-25" src="../images/stars1.jpg"> </div>
</div>
<div class="xs-4-wide">
<img class="fit-width opacity-40" src="../images/stars1.jpg"> </div>
</div>
</div>

Control the image opacity by inserting class opacity-value and border radius by adding the border-radius-value class
