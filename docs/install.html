Getting Started

Install

NPM install

Constellation UI is usable by the node package manager. The node package manager is the most straight forward to gain access to all functionality. Run npm init on your folder and then:

> npm i constellation-ui

In HTML files add the following code in the head tags

<link rel="stylesheet" type="text/css" href="node_modules/constellation-ui/css/index.css">

If you want to utilise the dropdown feature, you must add the following link also

<script src="node_modules/constellation-ui/index.js"></script>
