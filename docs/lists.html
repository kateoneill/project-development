Lists

Top Ten or Shopping Lists, we use lists everywhere so why not integrate them to your website using Constellation UI

Sample Code
<div class="row gutter-12">
<div class="xs-2-wide">
<ul class="Roboto list-style-square">
<li > Big Dipper </li>
<li > Gemini </li>
<li > Orion </li>
</ul>
</div>
<div class="xs-2-wide">
<ul class="Roboto line-height-250">
<li > Big Dipper </li>
<li > Gemini </li>
<li > Orion </li>
</ul>
</div>
<div class="xs-4-wide">
<ul class="Roboto horizontal list-style-disc">
<li > Big Dipper </li>
<li > Gemini </li>
<li > Orion </li>
</ul>
</div>
</div>

Full control over the Lists by adding classes to change counter as in the first example, alter the line-height as in the second example or throw the book out the window and make your lists horizontal, There is also options to adjust margin and padding for the entire list as well as having the ability to fully customise around each l. JUst add classes to whichever element for margin-value, margin-top-value, margin-bottom-value, margin-right-value, margin-left-value and same for the padding.
