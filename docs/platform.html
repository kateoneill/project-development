Platform

Elevating your content is key to dark mode design. COnstellation UI utilises a unique platform component to give your content a much needed lift.

Sample Platform

Simplest platform

Sample Code
<div class="platform">

</div>

These docs are built using a variety of platforms, using varying grey shades to lift the content off the dark background.

Sample Platform with elevations

Platform with elevation-2

Sample Code
<div class="platform elevation-2">

</div>

Sample Platform with elevations

Platform with elevation-8 and outline

Sample Code
<div class="platform elevation-8 outline-black">

</div>

Elevation points can be between elevation-1 and elevation-11. Outlines are available in outline-black or outline-white
