Tables

To display organised data, why not choose a table.

Sample Table Styling

Simplest table

Heading 1 Heading 2 Heading 3
Info 1 Info 2 Info 3
Sample Code
<table class="border-2 border-collapse padding-20" style="color:white;">
<tr class="orange">
<th> Heading 1</th >
<th> Heading 2</th >
<th> Heading 3</th >
</tr>
<tr>
<td> Info 1</td >
<td> Info 2</td >
<td> Info 3</td >
</tr>
</table>

The table component offers an abundance of customisation, alter the width or any row to use a themes colour or grey variation. Adjust padding on entire table or individual cell. Want to get rid of the border on one cell put in eiher of the following classes: border-right-{value}, border-left-{value}, border-top-{value} or border-bottom-{value}. Include these on any th or td elements for full customisation
