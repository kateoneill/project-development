Getting Started

Themes

Dark Mode has strict colour contrast ratios that need to be met as advised by Material UI guidelines. Constellation has fleshed out 3 themes for use by developers

Big Dipper

Default theme

Orion

Lighter tones

Cassiopeia

Bright pop

How to use the themes

Simply add the theme name to your body class and integrate colours as listed above on alerts, buttons or table elements as follows. It defaults to use of big dipper so need to mention that

Sample Code
<body class="orion">
<body class="cassiopeia">
